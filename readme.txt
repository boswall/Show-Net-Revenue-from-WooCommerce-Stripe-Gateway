Shows the Stripe fee and net revenue for Woocommerce order listings in the admin area.

== Description ==
The WooCommerce Stripe Gateway adds extra information to each order about the fee taken and the remaining net revenue. However, it doesn\'t show this anywhere.

This plugin adds two columns to the WooCommerce Order list and below the Order totals on the Order page. Visible by shop admins only.
